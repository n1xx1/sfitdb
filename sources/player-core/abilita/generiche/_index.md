# Azioni Generiche delle Abilità

Le azioni generiche delle abilità sono azioni utilizzabili con più abilità.
Quando usi un’azione generica delle abilità, a seconda della situazione puoi
usare il modificatore di una qualsiasi delle abilità che la elencano tra le
proprie azioni
