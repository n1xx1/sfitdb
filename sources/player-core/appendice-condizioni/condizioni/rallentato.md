# Rallentato

Hai meno azioni. Rallentato include sempre un valore. Quando recuperi le tue
azioni all’inizio del tuo turno, riduci quel numero di azioni del tuo valore di
rallentato. Poiché rallentato ha effetto all’inizio del turno, se diventi
rallentato durante il tuo turno, non perdi nessun’azione nell’immediato.
