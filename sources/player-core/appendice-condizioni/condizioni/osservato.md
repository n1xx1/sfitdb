# Osservato

Qualsiasi cosa sia in piena vista è osservato da te. Se una creatura cerca di
evitare l’individuazione, per esempio usando [Furtività](/abilita/furtivita) per
[Nascondersi](/azioni/nascondersi), può diventare
[nascosta](/condizioni/nascosto) o
[non individuata](/condizioni/non-individuato) anziché osservata. Se hai un
altro
[senso preciso](/regole/player-core/8-giocare/percezione-e-individuazione#sensi-precisi)
al posto della vista, o in aggiunta ad essa, puoi essere in grado di osservare
una creatura o un oggetto usando quel senso. Puoi osservare una creatura solo
con sensi precisi. Quando [Individui](/azioni/individuare) una creatura usando
solo sensi imprecisi, essa rimane nascosta anziché osservata.
