# Arrestare una Caduta :r:

**Innesco** Cadi.

**Requisiti** Hai una Velocità di volare.

---

Effettui una prova di Acrobazia per rallentare la tua caduta. La CD è in genere
15, ma può aumentare in caso di turbolenza dell’aria o altre circostanze.

**Successo** Non subisci danni dalla caduta.
