# Pronto Soccorso Avanzato - Talento 7

**Prerequisiti** maestro in [Medicina](/abilita/medicina)

---

Usi la tua preparazione medica per alleviare le malattie o placare le paure.
Quando usi Medicina per
[Prestare Primo Soccorso](/azioni/prestare-primo-soccorso), invece di
Stabilizzare un personaggio o Fermare il Sanguinamento, puoi ridurre la
condizione [spaventato](/condizioni/spaventato) o
[nauseato](/condizioni/nauseato) di un alleato di 2 o rimuovere interamente una
di queste condizioni con un successo critico. Puoi rimuovere solo una condizione
alla volta. La CD della prova di Medicina di solito è la CD dell’effetto che ha
causato la condizione.
