# Intrattenitore Virtuoso - Talento 1

**Prerequisiti** addestrato in [Esibizione](/abilita/esibizione)

---

Hai un talento eccezionale in un certo tipo di esibizione. Ottieni bonus di
circostanza +1 quando tieni quel tipo di esibizione. Se sei maestro in
Esibizione, il bonus aumenta a +2. Seleziona una delle seguenti specializzazioni
e applica il bonus quando effettui una prova di Esibizione del tipo scelto. Se
la possibilità di applicare la specialità è incerta, è il GM a decidere.

| Specialità              | Esempi                                                |
| ----------------------- | ----------------------------------------------------- |
| Canto                   | Ballata, canto, marinaresco, pop, rock                |
| Commedia                | Barzellette, gag comiche, routine di cabaret          |
| Cordofoni               | Chitarra, bobina corale, theremin                     |
| Danza                   | Classica, di sala, interpretativa, pop o tradizionale |
| Oratoria                | Epica, narrativa, ode, poesia                         |
| Recitazione             | Dirette streaming, reality, teatro                    |
| Sintetizzatori          | Plancia di sintetizzatori, consolle dj                |
| Strumenti a fiato       | Corno, flauto, a canne                                |
| Strumenti a percussione | Batteria, campane, chimes, xilofono                   |
| Tastiere                | Keytar, organo, piano                                 |
