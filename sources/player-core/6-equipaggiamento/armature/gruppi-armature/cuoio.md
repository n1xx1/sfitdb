# Cuoio

Lo spesso secondo strato dell’armatura disperde la forza dell’impatto per
ridurre il danno contundente. Ottieni resistenza ai danni contundenti pari a 1 +
il valore di resiliente dell'armatura per le armature medie, oppure 2 + il
valore di resiliente dell’armatura per quelle pesanti.
