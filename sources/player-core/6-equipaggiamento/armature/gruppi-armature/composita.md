# Composita

Le numerose parti sovrapposte di quest’armatura ti proteggono dagli attacchi
perforanti. Ottieni resistenza ai danni perforanti pari a 1 + il valore di
resiliente dell'armatura per le armature medie, oppure 2 + il valore di
resiliente dell’armatura per quelle pesanti.
