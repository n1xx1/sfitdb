# Arcaica

Questa armatura è realizzata utilizzando metodi e materiali tradizionali. Tutte
le armature di Pathfinder hanno il tratto arcaico. Le rune dell’armatura (vedi
_Pathfinder GM Core_) funzionano normalmente con un’armatura arcaica.
