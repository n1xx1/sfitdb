# Chip Dati - Oggetto 0

**Prezzo** 2 crediti

**Volume** —

---

Un chip dati contiene memoria sufficiente per un testo religioso, un album
fotografico, un diario personale o simili. Puoi visualizzare le informazioni con
un computer, come quello nella tua
[unità di comunicazione](/equipaggiamento/unita-di-comunicazione).
