# Kit Medico - Oggetto 1+

**Mani** 1 o 2; **Volume** 1

---

Questi contengono analgesici, bende, garze sterili e strumenti per sutura
necessari per le prove di [Medicina](/abilita/medicina) per
[Prestare Primo Soccorso](/azioni/prestare-primo-soccorso),
[Curare Ferite](/azioni/curare-ferite),
[Curare Malattie](/azioni/curare-malattie) o
[Curare Veleno](/azioni/curare-veleno).
