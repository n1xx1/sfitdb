# Coprire Tracce

Copri le tue tracce, muovendoti fino a metà della tua
[Velocità di viaggio](/regole/TODO-travel-speed). Non c’è bisogno di effettuare
una prova di Sopravvivenza per coprire le tue tracce; è chi le segue a dover
superare una prova di Sopravvivenza contro la tua CD di Sopravvivenza, se è
superiore rispetto alla normale CD per [Seguire Tracce](/azioni/seguire-tracce).

A volte potresti Coprire Tracce durante un incontro. In questo caso Coprire
Tracce è un’azione singola e non ha il tratto
[esplorazione](/tratti/esplorazione).
