# Hackerare

**Requisiti** Hai un
[kit di strumenti di hackeraggio](/equipaggiamento/kit-di-strumenti-di-hackeraggio).

---

Cerchi di accedere, controllare o modificare un sistema protetto attivo. Nella
maggior parte dei casi questo tentativo è diretto contro un sistema informatico
locale a cui accedi direttamente o indirettamente usando il kit di strumenti di
hackeraggio. Effettua una prova di Computer per determinare se puoi accedere al
sistema preso di mira. I computer con sistemi avanzati potrebbero richiedere più
successi per essere violati.

**Successo Critico** accedi al sistema, oppure ottieni due successi verso
l’accesso di un sistema che richiede più di un successo. Non lasci tracce della
tua manomissione.

**Successo** accedi al sistema, oppure ottieni un successo verso l’accesso di un
sistema che richiede più di un successo.

**Fallimento** non riesci ad accedere al sistema né a ottenere successi verso
l’accesso di un sistema che richiede più di un successo. Alcuni sistemi
informatizzati potrebbero attivare misure di sicurezza in caso di fallimento.

**Fallimento Critico** attivi protocolli di sicurezza particolarmente radicati o
ti blocchi completamente fuori dal sistema. Essere bloccato fuori impedisce di
tentare un altra prova per accedervi per l’ora successiva.
